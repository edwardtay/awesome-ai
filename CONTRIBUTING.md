# Contributing to Awesome AI APIs

Thank you for your interest in contributing! This list aims to be the most comprehensive resource for AI APIs, tools, and platforms.

## Guidelines

### Adding a new tool

- Make sure the tool is actively maintained and available.
- Add it in the appropriate category, in alphabetical order within the section.
- Use this format: `- [Name](URL) - One-line description ending with a period.`
- Keep descriptions concise (1-2 sentences max) and focus on what makes the tool distinctive.
- Do not use promotional language.

### Quality standards

- The tool should be actively maintained (updated within the last 12 months).
- It should have a working website or repository.
- It should be relevant to building AI-powered applications.

### Submitting a pull request

1. Fork the repository.
2. Add your entry following the format above.
3. Make sure your entry is in the correct category.
4. Submit a pull request with a clear title (e.g., "Add ToolName to Category").
5. Include a brief note about why the tool is awesome.

### Suggesting a new category

If your tool doesn't fit any existing category, open an issue to discuss adding a new one.

## Code of Conduct

Be respectful and constructive. We're building a useful resource together.
